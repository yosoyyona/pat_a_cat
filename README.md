## Instruction
This game is a kind of whack a mole. When the cat(Lilu) comes out of the box, type the keyboard buttons in the right direction(‚Üê‚Üë‚Üì‚Üí), so you can pat Lilu and get 1 point at a time. If you tab keys when she is in the box, you gonna lose 1 point, so watch out!
